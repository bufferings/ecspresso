1.2 インストール方法

バイナリからのインストール

ecspressoは、Go言語で書かれたシングルバイナリアプリケーションです。ビルド済みバイナリを使用してインストールできます。

実行可能ファイルの設定

$ chmod +x ecspresso
$ mv ecspresso /usr/local/bin/

ソースからのビルド

Go言語の開発環境がある場合は、ソースからecspressoをビルドすることもできます。

$ go install github.com/kayac/ecspresso/v2/cmd/ecspresso@latest

AWS認証の設定

ecspressoはAWS SDKを使用してAWSリソースにアクセスします。以下のいずれかの方法でAWS認証情報を設定してください。

AWS CLIの認証情報を使用

すでにAWS CLIが設定されている場合は、その認証情報がecspressoでも使用されます。

$ aws configure
AWS Access Key ID [None]: [アクセスキー]
AWS Secret Access Key [None]: [シークレットキー]
Default region name [None]: ap-northeast-1
Default output format [None]: json

環境変数の使用

export AWS_ACCESS_KEY_ID=[アクセスキー]
export AWS_SECRET_ACCESS_KEY=[シークレットキー]
export AWS_REGION=ap-northeast-1

IAMロールの使用（EC2、ECSタスク、Lambda内）

AWS環境内で実行する場合は、インスタンスプロファイルやタスクロールを使用できます。この場合、明示的な認証情報設定は不要です。

必要なIAM権限

ecspressoを使用するには、以下のAWSサービスへのアクセス権限が必要です：

  • Amazon ECS: サービス、タスク定義、タスクの管理
  • AWS CodeDeploy: Blue/Greenデプロイを使用する場合
  • Amazon CloudWatch Logs: ログの表示
  • AWS Auto Scaling: Auto Scalingの設定を管理する場合
  • AWS Systems Manager Parameter Store: パラメータを参照する場合
  • AWS Secrets Manager: シークレットを参照する場合
  • Elastic Load Balancing: ロードバランサーを使用する場合
  • AWS Service Discovery: サービスディスカバリーを使用する場合
  • VPC Lattice: VPC Latticeを使用する場合

バージョンの確認

インストールが正常に完了したことを確認するには、バージョン情報を表示します。

$ ecspresso version

初期設定

ecspressoを初めて使用する場合は、以下のコマンドで初期設定ファイルを生成できます：

$ ecspresso init --region ap-northeast-1 --service your-service-name --cluster your-cluster-name

このコマンドは、カレントディレクトリにecspresso.ymlファイルを生成します。このファイルを編集して、サービスの設定を行います。

設定ファイルの構造

ecspressoの設定ファイル（ecspresso.yml）には、以下の主要な設定項目があります：

region: ap-northeast-1
cluster: your-cluster-name
service: your-service-name
service_definition: ecs-service-def.json
task_definition: ecs-task-def.json
timeout: 10m
plugins:
  - name: tfstate
    config:
      path: terraform.tfstate

各項目の説明：

  • region: AWSリージョン
  • cluster: ECSクラスター名
  • service: ECSサービス名
  • service_definition: サービス定義ファイルのパス（JSON/YAML）
  • task_definition: タスク定義ファイルのパス（JSON/YAML）
  • timeout: 操作のタイムアウト時間
  • plugins: 使用するプラグインの設定
