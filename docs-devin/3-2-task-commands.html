3.2 タスク関連コマンド

run コマンド

ECSタスクを実行します。一時的なバッチ処理やデータベースマイグレーションなどに便利です。

基本的な使い方

$ ecspresso run [オプション]

主なオプション

オプション 説明 デフォルト値
--dry-run 実際の変更を行わずに実行内容を表示 false
--task-def 使用するタスク定義ファイルのパス 設定ファイルの値
--revision 使用するタスク定義のリビジョン 0（最新）
--count 起動するタスクの数 1
--launch-type 起動タイプ（EC2/FARGATE） 設定による
--network-configuration ネットワーク設定のJSONファイル -
--overrides タスク定義のオーバーライドを含むJSONファイル -
--container コマンドを実行するコンテナ名 -
--command 実行するコマンド（JSON配列形式） -
--env 環境変数（KEY=VALUE形式） -
--wait タスクの完了を待つ false
--latest-task-definition 最新のタスク定義を使用 false
--no-wait-task-running タスクの実行開始を待たない false

使用例

$ ecspresso run --task-def=custom-task.json --command='["bin/migrate", "up"]' --wait

カスタムタスク定義を使用してマイグレーションコマンドを実行し、完了を待ちます。

tasks コマンド

ECSタスクの一覧表示や停止を行います。

基本的な使い方

$ ecspresso tasks [オプション]

主なオプション

オプション 説明 デフォルト値
--id 特定のタスクIDを指定 -
--find タスクIDの一部で検索 -
--stop タスクを停止 false
--trace タスクのトレース情報を表示 false
--output 出力形式（json/table） table
--status フィルタリングするタスクのステータス RUNNING

使用例

$ ecspresso tasks --find=migration --stop

IDに「migration」を含むタスクを検索して停止します。

exec コマンド

実行中のECSタスク内でコマンドを実行します。デバッグやトラブルシューティングに便利です。

基本的な使い方

$ ecspresso exec [オプション]

主なオプション

オプション 説明 デフォルト値
--id 特定のタスクIDを指定 -
--container コマンドを実行するコンテナ名 -
--command 実行するコマンド -
--port-forward ポートフォワーディング（ローカル:リモート） -
--interactive 対話モードで実行 true

使用例

$ ecspresso exec --command="/bin/bash"

実行中のコンテナでbashシェルを起動します。

$ ecspresso exec --port-forward=8080:80

コンテナの80番ポートをローカルの8080番ポートにフォワードします。
