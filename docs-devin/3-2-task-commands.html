run

ECSタスクを実行します。

オプション

オプション 説明 デフォルト値
--dry-run 実際の変更を行わずに実行内容を表示 false
--task-def 実行するタスク定義ファイル -
--wait/--no-wait タスクが完了するまで待機するかどうか true
--overrides タスクオーバーライドのJSON文字列 -
--overrides-file タスクオーバーライドのJSONファイルパス -
--skip-task-definition 新しいタスク定義の登録をスキップ false
--count 実行するタスクの数（最大10） 1
--watch-container 終了コードを監視するコンテナ名 -
--latest-task-definition 新しいタスク定義を登録せずに最新のタスク定義を使用 false
--propagate-tags タスクに伝播するタグ（SERVICE または TASK_DEFINITION） -
--tags タスクのタグ（形式: KeyFoo=ValueFoo,KeyBar=ValueBar） -
--wait-until タスクのステータスが到達するまで待機（running または stopped） stopped
--revision --skip-task-definitionを使用する場合のタスク定義のリビジョン 0
--client-token リクエストを識別する一意のトークン（べき等性に役立つ） -
--ebs-delete-on-termination/--no-ebs-delete-on-termination タスクが停止したときにEBSボリュームを削除するかどうか true

使用例

$ ecspresso run
$ ecspresso run --overrides='{"containerOverrides":[{"name":"app","command":["echo","hello"]}]}'
$ ecspresso run --count 5
$ ecspresso run --wait-until running
$ ecspresso run --tags "Key1=Value1,Key2=Value2"

tasks

サービス内のタスクまたは同じファミリーを持つタスクを一覧表示します。

オプション

オプション 説明 デフォルト値
--status フィルタリングするタスクのステータス（RUNNING, STOPPED, ...） -
--id 特定のタスクIDを指定 -
--output 出力形式（table, json, tsv） table
--find-by-family サービスではなくタスク定義ファミリーでタスクを検索 false
--stop 一覧表示されたタスクを停止 false
--force 確認なしでタスクを停止 false

使用例

$ ecspresso tasks
$ ecspresso tasks --status RUNNING
$ ecspresso tasks --output json
$ ecspresso tasks --stop --force

exec

タスク上でコマンドを実行します。

オプション

オプション 説明 デフォルト値
--id 特定のタスクIDを指定 -
--container コマンドを実行するコンテナ名 -
--command 実行するコマンド -
--interactive/--no-interactive 対話モードを有効/無効にする true
--port-forward ポートフォワーディングを有効にする（形式: local:remote） -

使用例

$ ecspresso exec --command /bin/bash
$ ecspresso exec --id task-id --container app --command "ls -la"
$ ecspresso exec --no-interactive --command "echo hello"
$ ecspresso exec --port-forward 8080:80

register

タスク定義を登録します。

オプション

オプション 説明 デフォルト値
--dry-run 実際の変更を行わずに実行内容を表示 false
--task-def 登録するタスク定義ファイル -

使用例

$ ecspresso register
$ ecspresso register --task-def custom-task-def.json
$ ecspresso register --dry-run

deregister

タスク定義の登録を解除します。

オプション

オプション 説明 デフォルト値
--dry-run 実際の変更を行わずに実行内容を表示 false
--revision 登録解除するタスク定義のリビジョン 0 (現在のリビジョン)

使用例

$ ecspresso deregister
$ ecspresso deregister --revision 123
$ ecspresso deregister --dry-run
