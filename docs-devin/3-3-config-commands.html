init

既存のECSサービスから設定ファイルを作成します。

オプション

オプション 説明 デフォルト値
--region AWSリージョン -
--cluster ECSクラスター名 -
--service ECSサービス名 -
--config 生成する設定ファイルのパス ecspresso.yml
--task-definition-path 生成するタスク定義ファイルのパス ecs-task-def.json
--service-definition-path 生成するサービス定義ファイルのパス ecs-service-def.json
--appspec-path 生成するAppSpecファイルのパス -
--assume-role-arn 引き受けるロールのARN -
--format 出力形式（json または yaml） json

使用例

$ ecspresso init --region ap-northeast-1 --cluster my-cluster --service my-service
$ ecspresso init --region ap-northeast-1 --cluster my-cluster --service my-service --format yaml
$ ecspresso init --region ap-northeast-1 --cluster my-cluster --service my-service --appspec-path appspec.yaml

diff

タスク定義、サービス定義と現在実行中のサービスおよびタスク定義の差分を表示します。

オプション

オプション 説明 デフォルト値
--unified 統一された差分形式を使用 false
--task-def 比較するタスク定義ファイル -
--compare-task-definition タスク定義の比較を有効/無効にする true
--compare-service サービス定義の比較を有効/無効にする true

使用例

$ ecspresso diff
$ ecspresso diff --unified
$ ecspresso diff --task-def custom-task-def.json
$ ecspresso diff --compare-task-definition=false

verify

設定内のリソースを検証します。

オプション

オプション 説明 デフォルト値
--task-def 検証するタスク定義ファイル -
--secrets シークレットの存在を検証 true
--cache 検証結果をキャッシュ true
--save-stopped 停止したタスクの検証結果を保存 false

使用例

$ ecspresso verify
$ ecspresso verify --task-def custom-task-def.json
$ ecspresso verify --secrets=false
$ ecspresso verify --cache=false

render

設定、サービス定義、またはタスク定義ファイルを標準出力にレンダリングします。

オプション

オプション 説明 デフォルト値
--config 設定ファイルをレンダリング false
--service-definition サービス定義をレンダリング false
--task-definition タスク定義をレンダリング false
--task-def レンダリングするタスク定義ファイル -

使用例

$ ecspresso render --config
$ ecspresso render --service-definition
$ ecspresso render --task-definition
$ ecspresso render --task-definition --task-def custom-task-def.json

status

サービスのステータスを表示します。

オプション

オプション 説明 デフォルト値
--events 表示するイベントの数 10

使用例

$ ecspresso status
$ ecspresso status --events 20

wait

サービスが安定するまで待機します。

オプション

オプション 説明 デフォルト値
--timeout タイムアウト時間 設定ファイルの値

使用例

$ ecspresso wait
$ ecspresso wait --timeout 5m

revisions

タスク定義のリビジョンを表示します。

オプション

オプション 説明 デフォルト値
--output 出力形式（table, json, tsv） table

使用例

$ ecspresso revisions
$ ecspresso revisions --output json

appspec

CodeDeployのAppSpec YAMLを標準出力に出力します。

オプション

オプション 説明 デフォルト値
--task-definition 使用するタスク定義ARN -

使用例

$ ecspresso appspec
$ ecspresso appspec --task-definition arn:aws:ecs:region:account:task-definition/family:revision
