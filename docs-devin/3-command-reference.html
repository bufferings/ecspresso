このセクションでは、ecspressoで使用できるすべてのコマンドとそのオプションについて詳しく説明します。コマンドは機能別に分類されており、それぞれのコマンドの使用方法、オプション、実行例を提供しています。

ecspressoのコマンドは大きく分けて、デプロイ関連コマンド、タスク関連コマンド、設定・検証コマンドの3つのカテゴリに分類されます。各カテゴリのコマンドの詳細については、対応するサブセクションを参照してください。

共通オプション

すべてのecspressoコマンドで使用できる共通オプションは以下の通りです：

オプション 説明 デフォルト値
--config 設定ファイルのパス ecspresso.yml
--envfile 環境変数ファイルのパス -
--debug デバッグログを有効にする false
--ext-str Jsonnet用の外部文字列値 -
--ext-code Jsonnet用の外部コード値 -
--assume-role-arn 引き受けるロールのARN -
--timeout タイムアウト時間 設定ファイルの値
--filter-command フィルターコマンド -
--color/--no-color カラー出力を有効/無効にする true
--log-format ログフォーマット（text/json） text

各コマンドの詳細なオプションと使用例については、対応するサブセクションを参照してください。
