v2.5.0 (2025-05-10)

ソースコード分析に基づく主な変更点：

timeline title v2.5.0の主な機能追加 section 機能強化 外部プラグインシステム : plugin.go deploy --wait-until : deploy.go CodeDeployの設定拡張 : config.go section 内部改善 ロガーをslogに変更 : log/slog JSON形式サポート : cli.go 検証モジュール改善 : verify.go
  • deploy --wait-untilオプションの追加 - deploy.goのDeployOption構造体にWaitUntilフィールドが追加され、デプロイ後の待機条件をより細かく制御できるようになりました。
  • 外部プラグインシステムの追加 - plugin.goにfindExternalPlugin関数が追加され、ecspresso-plugin-*という名前のバイナリを検索して実行する機能が実装されました。
  • CodeDeployの設定にdeployment_config_nameの追加 - config.goのCodeDeploy構造体にDeploymentConfigNameフィールドが追加され、デプロイメント設定の柔軟性が向上しました。
  • ロガーをlog/slogに切り替え - 標準ライブラリのlog/slogパッケージを使用するように変更され、構造化ログ出力が可能になりました。
  • ログと出力のJSON形式サポートの追加 - cli.goにWriteOutput関数が拡張され、JSON形式での出力が可能になりました。
  • コンテキスト化された状態による検証モジュールの改善 - verify.goのverifyState構造体が拡張され、検証状態のキャッシュと再利用が改善されました。
  • AppSpecのJSON形式サポートの追加 - appspec/spec.goにMarshalJSONメソッドが追加され、AppSpecをJSON形式で出力できるようになりました。
  • サプライチェーン攻撃を軽減するために外部アクションをピン留め - GitHub Actionsの設定ファイルが更新され、外部アクションのバージョンが固定されました。
  • 依存関係の更新 - AWS SDK for Go v2などの依存関係が最新バージョンに更新されました。
