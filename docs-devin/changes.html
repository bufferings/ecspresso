変更履歴

v2.5.0 (2025-05-10)

  • サプライチェーン攻撃を軽減するために外部アクションをピン留め
  • github.com/golang-jwt/jwt/v4 を 4.5.1 から 4.5.2 にアップデート
  • github.com/golang-jwt/jwt/v5 を 5.2.1 から 5.2.2 にアップデート
  • golang.org/x/crypto を 0.32.0 から 0.35.0 にアップデート
  • 機能リクエスト: 新しいオプション: サービスの安定を待つのではなく、サービスのデプロイメントを待つ
  • deploy --wait-until オプションを追加
  • 内部使用のために WaitUntil を設定
  • 16のアップデートを含む1つのディレクトリ全体で aws-sdk-go-v2 グループをアップデート
  • 外部プラグインを追加
  • CodeDeploy の設定に deployment_config_name を追加
  • ロガーを log/slog に切り替え
  • ログと出力の JSON フォーマットサポートを追加
  • コンテキストのキャンセル処理のために time.Sleep を sleepContext に置き換え
  • リファクタリング: コンテキスト化された状態を使用して検証モジュールを改善
  • 機能: AppSpec の JSON サポートを追加
