ecspresso について

ecspressoは、AWS Elastic Container Service (ECS) のコンテナデプロイと管理を簡素化するコマンドラインツールです。宣言的な設定ファイルを使用して、ECSサービスのデプロイ、スケーリング、ロールバックなどの操作を自動化します。

flowchart TB User[ユーザー] -->|設定ファイル| Ecspresso[ecspresso] Ecspresso -->|API呼び出し| AWS[AWS API] AWS --> ECS[Amazon ECS] AWS --> CD[CodeDeploy] AWS --> ECR[Amazon ECR] AWS --> CW[CloudWatch] AWS --> IAM[IAM]

主な機能

  • ECSサービスとタスク定義の管理
  • デプロイとロールバック
  • ワンオフタスクの実行
  • コンテナ内でのコマンド実行
  • リソース設定の検証
  • 構成の差分表示
