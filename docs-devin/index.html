Overview

ecspresso is a command-line tool designed to simplify and automate deployments to Amazon Elastic Container Service (ECS). It serves DevOps engineers, SRE teams, and cloud engineers who manage containerized applications on AWS.

Key Features

  • Deploy, scale, and rollback ECS services using declarative configuration files
  • Compare local configurations with deployed resources to identify drift
  • Execute one-off tasks and commands within ECS containers
  • Verify resource configurations before deployment
  • Support advanced deployment strategies like blue/green deployments via AWS CodeDeploy

Project Organization

ecspresso consists of several core systems:

  • CLI Interface: The entry point for user commands, providing a clean interface to the underlying functionality.
  • Configuration Management: Handles loading, parsing, and validating configuration files.
  • Deployment Operations: Manages service deployments, rollbacks, and updates.
  • Task Operations: Handles running, stopping, and executing commands in ECS tasks.
  • Resource Verification: Validates AWS resources before deployment.
  • Plugin System: Extends functionality through plugins for Terraform state, CloudFormation, SSM, and Secrets Manager.
  • AWS SDK Integration: Interacts with various AWS services via the AWS SDK for Go v2.
