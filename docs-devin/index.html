はじめに

ecspressoは、Amazon ECS（Elastic Container Service）へのデプロイメントを簡素化および自動化するためのコマンドラインツールです。DevOpsエンジニア、SREチーム、AWSでコンテナ化されたアプリケーションを管理するクラウドエンジニアを対象としています。

このツールは、Go言語で実装され、AWSのSDKを使用してECSサービスとの対話を行います。主要なロジックはecspresso.goファイルに定義されたApp構造体を中心に構築されています。

主な機能

  • 宣言的な設定ファイルを使用したECSサービスのデプロイ、スケーリング、ロールバック
  • ローカル設定とデプロイされたリソースの比較によるドリフト検出
  • ECSコンテナ内での一時的なタスクの実行とコマンド実行
  • デプロイ前のリソース構成の検証
  • AWS CodeDeployを介したブルー/グリーンデプロイメント戦略のサポート

主要コンポーネント

ecspressoは以下の主要コンポーネントで構成されています：

flowchart TD A[App構造体] --> B[CLI] A --> C[設定管理] A --> D[デプロイ操作] A --> E[タスク操作] A --> F[リソース検証] A --> G[差分比較] A --> H[プラグインシステム] C --> I[YAML/JSON/Jsonnet] D --> J[ECSデプロイ] D --> K[CodeDeployデプロイ] E --> L[タスク実行] E --> M[コンテナ内コマンド実行] F --> N[IAM/ECS/ECR/VPC検証] H --> O[tfstate] H --> P[CloudFormation] H --> Q[SSM] H --> R[Secrets Manager]
  • App構造体：ecspresso.goで定義された中心的なオーケストレーター。AWS SDKクライアントと設定への参照を含み、すべての主要な操作を実装。
  • CLI：cli.goでコマンドライン引数を解析し、適切なハンドラーにディスパッチ。
  • 設定管理：config.goでecspresso.ymlからの設定値を管理。
  • デプロイ操作：deploy.goでECSサービスのデプロイロジックを実装。
  • リソース検証：verify.goでリソースが正しく構成されていることを確認。
  • 差分比較：diff.goでローカル定義と実際のリソースの違いを表示。
  • タスク管理：tasks.goおよびrun.goでタスクの実行と管理を担当。
  • プラグインシステム：plugin.goで機能拡張のための柔軟なプラグインシステムを提供。

最新アップデート

ソースコード分析に基づく最新バージョン（v2.5.0）の主な機能と改善点：

  • deploy --wait-untilオプションの追加による細かなサービスデプロイ制御
  • 外部プラグインシステムの追加によるカスタム拡張の容易化
  • CodeDeployの設定にdeployment_config_nameオプションの追加
  • Goの標準ロガーlog/slogへの移行による改善されたログ機能
  • ログと出力のJSON形式サポートの追加によるプログラム的処理の強化
  • コンテキスト化された状態による検証モジュールの改善
