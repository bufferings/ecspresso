概要

ecspresso は Amazon Elastic Container Service (ECS) へのデプロイを簡素化・自動化するためのコマンドラインツールです。AWS上でコンテナ化されたアプリケーションを管理するDevOpsエンジニア、SREチーム、クラウドエンジニア向けに設計されています。

主な機能

  • 宣言的な設定ファイルを使用した ECS サービスのデプロイ、スケーリング、ロールバック
  • ローカル構成と実際にデプロイされたリソースを比較して差異を特定
  • ECS コンテナ内での一時的なタスクやコマンドの実行
  • デプロイ前のリソース構成の検証
  • AWS CodeDeploy を通じたブルー/グリーンデプロイメントなどの高度なデプロイ戦略のサポート

プロジェクト構成

ecspresso は以下のコアシステムで構成されています：

  • CLI インターフェース: ユーザーコマンドのエントリーポイントとなり、基盤となる機能へのクリーンなインターフェースを提供します。
  • 設定管理: 設定ファイルの読み込み、解析、検証を処理します。
  • デプロイ操作: サービスのデプロイ、ロールバック、更新を管理します。
  • タスク操作: ECS タスクの実行、停止、コマンド実行を処理します。
  • リソース検証: デプロイ前の AWS リソースを検証します。
  • プラグインシステム: Terraform state、CloudFormation、SSM、Secrets Manager 用のプラグインを通じて機能を拡張します。
  • AWS SDK 統合: AWS SDK for Go v2 を介して様々な AWS サービスと連携します。

最新アップデート (v2.5.0)

最新バージョンには、以下の新機能と改善が含まれています：

  • deploy --wait-until オプションの追加によるサービスデプロイの詳細な制御
  • 機能拡張のための外部プラグインシステムの追加
  • CodeDeploy の設定に deployment_config_name の追加
  • ロガーを log/slog に切り替え
  • ログと出力の JSON フォーマットサポートの追加
  • コンテキスト化された状態による検証モジュールの改善
  • AppSpec の JSON サポートの追加
