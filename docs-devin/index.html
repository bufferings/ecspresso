ecspresso について

ecspressoは、AWS Elastic Container Service (ECS) のコンテナデプロイと管理を簡素化するコマンドラインツールです。宣言的な設定ファイルを使用して、ECSサービスのデプロイ、スケーリング、ロールバックなどの操作を自動化します。

主な機能

  • ECSサービスとタスク定義の管理（デプロイ、ロールバック、更新）
  • 複数のデプロイ戦略（ECSローリングアップデート、CodeDeployによるBlue/Green）
  • ワンオフタスクの実行と管理
  • コンテナ内でのコマンド実行（exec）
  • リソース設定の検証と差分表示
  • Auto Scalingの設定と管理
  • プラグインシステムによる拡張性（Terraform、CloudFormation、SSM、Secrets Manager）
