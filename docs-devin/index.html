ecspresso ドキュメント

ecspressoは、Amazon Elastic Container Service (ECS) へのデプロイを簡素化・自動化するためのコマンドラインツールです。宣言的な設定ファイルを使用して、ECSサービスのデプロイ、スケーリング、ロールバックなどの操作を簡単に行うことができます。

graph TD
    A[設定ファイル] --> B[ecspresso]
    B --> C[AWS ECS]
    B --> D[AWS CodeDeploy]
    C --> E[コンテナサービス]
    D --> F[Blue/Greenデプロイ]

主な機能

  • 宣言的な設定ファイルによるECSサービスのデプロイ
  • ローカル設定と実際のリソースの差分比較
  • タスクの実行とコンテナ内でのコマンド実行
  • デプロイ前のリソース検証
  • AWS CodeDeployを使用したブルー/グリーンデプロイのサポート
