プラグイン

ecspressoはテンプレート関数とJsonnetネイティブ関数を拡張するプラグインをサポートしています。

tfstate

tfstateプラグインはtfstateとtfstatefテンプレート関数を導入します。

# ecspresso.yml
region: ap-northeast-1
cluster: default
service: test
service_definition: ecs-service-def.json
task_definition: ecs-task-def.json
plugins:
  - name: tfstate
    config:
      url: s3://my-bucket/terraform.tfstate
      # または path: terraform.tfstate    # ローカルファイルへのパス
# ecs-service-def.json
{
  "networkConfiguration": {
    "awsvpcConfiguration": {
      "subnets": [
        "{{ tfstatef `aws_subnet.private['%s'].id` `az-a` }}"
      ],
      "securityGroups": [
        "{{ tfstate `data.aws_security_group.default.id` }}"
      ]
    }
  }
}

{{ tfstate "resource_type.resource_name.attr" }}はtfstate内のリソースの属性値に展開されます。

{{ tfstatef "resource_type.resource_name['%s'].attr" "index" }}は{{ tfstatef "resource_type.resource_name['index'].attr" }}と同様です。この関数は環境変数を使用してリソースアドレスを構築するのに役立ちます。

tfstate Jsonnet関数

local tfstate = std.native('tfstate');
{
  networkConfiguration: {
    awsvpcConfiguration: {
      subnets: [
        tfstate('aws_subnet.private["%s"].id' % 'az-z'),
        tfstate(std.format('aws_subnet.private["%s"].id', 'az-b')),
      ],
      securityGroups: [
        tfstate('data.aws_security_group.default.id'),
      ]
    }
  }
}

CloudFormation

cloudformationプラグインはcfn_outputとcfn_exportテンプレート関数を導入します。

# ecspresso.yml
# ...
plugins:
  - name: cloudformation

cfn_output StackName OutputKeyはStackName内のOutputKeyのOutputValueを検索します。

cfn_export ExportNameは名前でエクスポートされた値を検索します。

SSM Parameter Storeの検索

ssmテンプレート関数はAWS Systems Manager（SSM）Parameter Storeからパラメータを読み取ります。

{
  "string": "{{ ssm `/path/to/string` }}",
  "stringlist": "{{ ssm `/path/to/stringlist` 1 }}",
  "securestring": "{{ ssm `/path/to/securestring` }}"
}

Secrets Managerのシークレットの解決

secretsmanager_arnテンプレート関数はシークレット名からSecrets Managerのシークレット ARNを解決します。

  "secrets": [
    {
      "name": "FOO",
      "valueFrom": "{{ secretsmanager_arn `foo` }}"
    }
  ]

外部コマンドの実行

externalプラグインは任意の外部コマンドを実行する関数を導入します。

# ecspresso.yml
plugins:
  - name: external
    config:
      name: jq
      command: ["jq", "-n"]
      num_args: 1
      timeout: 5

configセクションは以下のパラメータを定義します：

  • name: テンプレート関数名
  • command: 外部コマンドと引数（配列）
    • コマンドは標準出力にJSON文字列または任意の文字列を返す必要があります。
  • num_args: 引数の数（オプション、デフォルト0）
  • parser: パーサータイプ "json"または "string"（オプション、デフォルト "json"）
  • timeout: コマンド実行タイムアウト秒（オプション、デフォルトはタイムアウトなし）
